Social Media

Social networks can be defined as web services that allow individuals to create a public (all users have access) or limited (only certain users have access) personal profile in the system, create a list of acquaintances, browse and search their own list of acquaintances Being a user of a large social network has certain advantages such as: a sense of connection with other individuals, meeting like-minded people, a new way of sharing life experiences and scientific discoveries, and managing the amount of personal data that will be displayed on social networks. , not enabled). The ability to manage the amount of personal data on the social network is certainly an advantage, because the user chooses which personal data he wants to disclose and thus protect his own privacy. Users are often unaware of the number of individuals to whom their personal information is available. User data is often not adequately protected, resulting in security incidents and misuse of personal data

Good and bad consequences of social media

Social networks are becoming a worldwide hit, providing users with an easier way to communicate, the opportunity to meet new people and share information with each other. The original social networks on the Internet did not have a large number of functionalities, so the number of security incidents was insignificant. Social networks have become an indispensable part of everyday life, and gather a large number of users in one place. The development and popularity of communication via the Internet are some of the reasons why social networks are constantly gaining in popularity. Man is a social being and it is only natural that he constantly expands the number of acquaintances, through direct contact, social networking sites or in any other way. Social networks allow individuals a kind of anonymity when contacting other people, even though personal information is printed on user profiles. Simplicity and ease of access to a stranger, based on common interests have never been easier.

Web development

Web development is the work involved in developing a Web site for the Internet.Web development can range from developing a simple single static page of plain text to complex Web-based Internet applications (Web apps), electronic businesses, and social network services. A more comprehensive list of tasks to which Web development commonly refers, may include Web engineering, Web design, Web content development, client liaison, client-side/server-side scripting, Web server and network security configuration, and e-commerce development.

Among Web professionals, "Web development" usually refers to the main non-design aspects of building Web sites: writing markup and coding There are three kinds of Web developer specialization: front-end developer, back-end developer, and full-stack developer. Front-end developers are responsible for behavior and visuals that run in the user browser, while back-end developers deal with the servers.
